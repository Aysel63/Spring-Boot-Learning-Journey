# Spring Boot 4, Spring 7 & Hibernate 

This repository is a comprehensive implementation of the **"Spring Boot 4, Spring 7 & Hibernate for Beginners"** masterclass by Chad Darby. It serves as a complete technical reference for building enterprise-level Java applications using modern standards.

* **Java:** 21 (LTS)
* **Frameworks:** Spring Boot 4, Spring Core 7
* **Data Management:** Hibernate 6, Spring Data JPA, MySQL
* **Security:** Spring Security 6 (Role-Based Access Control - RBAC)
* **Templating:** Thymeleaf & Bootstrap 5
* **Build Tool:** Maven
* **Version Control:** Git & GitHub

## üìÇ Curriculum & Implemented Modules
The project is structured according to the professional learning path, covering the following core areas:

### 1. Spring Boot Core & Fundamentals
* **Spring Boot 3 Overview:** Rapid application development and DevTools integration.
* **Inversion of Control (IoC) & Dependency Injection (DI):** Component scanning, Qualifiers, and Bean Scopes.

### 2. Data Access with Hibernate & JPA
* **CRUD Operations:** Implementing Create, Read, Update, and Delete logic with MySQL.
* **REST API Development:** Building high-performance RESTful services with JSON support and global exception handling.

### 3. Enterprise Security
* **Spring Security 6:** Implementing secure login/logout flows.
* **Authentication & Authorization:** Database-backed (JDBC) authentication and role-based access control (User, Manager, Admin).

### 4. Web Development & Validation
* **Spring MVC:** Developing server-side web applications using the Model-View-Controller pattern.
* **Form Validation:** Strict server-side validation using Hibernate Validator/Bean Validation API.
* **Thymeleaf Integration:** Dynamic content rendering with Bootstrap-styled frontends.

### 5. Advanced Hibernate Mappings & AOP
* **Entity Relationships:** One-to-One, One-to-Many, and Many-to-Many advanced mappings.
* **Aspect-Oriented Programming (AOP):** Decoupling cross-cutting concerns like logging and security.
